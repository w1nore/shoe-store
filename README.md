# ООО "Обувь" - Информационная система

Веб-приложение для управления магазином обуви с разделением прав доступа.

## Функционал

### Роли пользователей:

1. **Гость**
   - Просмотр списка товаров (без фильтрации, сортировки, поиска)

2. **Авторизованный клиент**
   - Просмотр списка товаров (без фильтрации, сортировки, поиска)

3. **Менеджер**
   - Просмотр товаров с фильтрацией, сортировкой, поиском
   - Просмотр заказов

4. **Администратор**
   - Все функции менеджера
   - Добавление/редактирование/удаление товаров
   - Добавление/редактирование/удаление заказов

## Установка и запуск

### Требования:
- Python 3.8 или выше
- pip

### Установка зависимостей:

```bash
pip install -r requirements.txt
```

### Запуск приложения:

```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000
